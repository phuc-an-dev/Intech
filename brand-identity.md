# Brand Identity Rules: Intech

This document defines the core brand identity and design principles for **Intech**. All future development, design, and content requests must adhere to these rules.

## 1. Core Concept
- **Brand Name**: Intech
- **Domain**: Course introduction, student mobility, and industry consultancy.
- **Tone of Voice**: Professional, Strong, and Reliable.
- **Slogan Proposal**: "Intech: Chạm tri thức, Kiến tạo tương lai." (Proposed)
- **Language**: Vietnamese (Primary).

## 2. Visual Identity (Design System)

### 2.1 Color Palette
- **Primary Blue (Deep Blue)**: `#002D62` (Strong, Trustworthy)
- **Secondary Cyan (Tech Cyan)**: `#00A3C1` (Innovative, Energetic)
- **Neutral Backgrounds**:
  - Light Gray: `#F4F7F9` (For sections)
  - Pure White: `#FFFFFF` (Main content)
- **Mode**: Light Mode ONLY.

### 2.2 Typography
- **Headings**: Strong Sans-serif (e.g., **Montserrat** Bold or **Inter** Extra Bold).
- **Body**: Clean Sans-serif (e.g., **Inter**).
- **Hierarchy**: Clear distinction with bold weights for headers.
- **Spacing**: Spacious `line-height` (1.6 - 1.8) and generous padding for readability.

### 2.3 UI Style
- **Design Philosophy**: Flat Design (No glassmorphism, no complex gradients).
- **Corners**: Fully Rounded / Pill-shaped buttons.
- **Icons**: **Outline style** only. 
- **CRITICAL**: **NO EMOJIS** in any UI element or content.
- **Layout**: **Mobile-First** approach. Spacious and breathable.
- **Images**: Use placeholders initially.

## 3. Brand Elements
- **Logo**: Wordmark style (Text-based).
- **Target Audience**: Students and young learners.

## 4. Tech Stack (Core)
- **Framework**: Next.js
- **Optimization**: SEO-friendly, fast loading, responsive.
